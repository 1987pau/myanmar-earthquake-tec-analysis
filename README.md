This script analyzes ionospheric anomalies triggered by the March 28, 2025, earthquake using GNSS Total Electron Content (TEC) data. It isolates the seismic impact by subtracting a quiet-time baseline from actual data to find the exact disturbance (dTEC). It also integrates Kp and Dst indices to rule out space weather interference, generating a synchronized 3-panel visualization of the event.
